
Example of coding up a decision tree learning problem in WillWaitProblem.java, using data from WillWait.data.txt (typed in from AIMA).

Some other datasets from UCI Machine Learning Repository (http://archive.ics.uci.edu/ml/index.php) for you to test your decision tree learner on. The ".names" file explains the data in the ".data" file.

- A discrete version of the Iris dataset
  https://archive.ics.uci.edu/ml/datasets/iris

- The Congressional Voting Records (``house votes 84'') dataset
  http://archive.ics.uci.edu/ml/datasets/Congressional+Voting+Records
